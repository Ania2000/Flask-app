# MovieDB Flask App

Prosta aplikacja webowa napisana w Pythonie z użyciem frameworka Flask oraz bazy danych SQLite.  
Aplikacja umożliwia wyświetlanie listy filmów, dodawanie nowych filmów oraz usuwanie wybranych pozycji.

## Funkcjonalności
- wyświetlanie listy filmów
- dodawanie nowych filmów
- usuwanie jednego lub wielu filmów


## Wymagania
- Python 3.10+
- Flask.
